## Backend Fast Api

this app is continue development...

### installation

first install fastapi

```
 python -m pip install fastapi
```

then install uvicorn

```
pip install uvicorn
```

## run the webserver

```
uvicorn myapi:app --reload
```

## access documentation

to access the docs, simply type :

```
http://127.0.0.1:8000/docs
```
